## Quick Start

If you're completely new to OrionECS, we recommend:

1. Read the [README](../../README.md) for a project overview
2. Start with **Tutorial 1: Your First ECS Project**
3. Work through the Beginner Path in order
4. Explore the [Examples](../../examples/) to see complete applications
5. Practice by building your own variations of the examples

## Tutorial Format

Each tutorial follows a consistent structure:

- **Learning Objectives**: What you'll learn
- **Prerequisites**: What you need to know first
- **Step-by-Step Guide**: Detailed instructions with code examples
- **Complete Code**: Full working example you can run
- **Try It Yourself**: Challenges and exercises
- **Troubleshooting**: Common issues and solutions
- **Next Steps**: Where to go from here

## Running Tutorial Code

All tutorials include runnable code examples in the `/tutorials` directory:

```bash
# Navigate to a tutorial example
cd tutorials/01-first-ecs-project

# Install dependencies (if needed)
npm install

# Run the tutorial code
npm start
```
